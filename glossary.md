# 术语表

## AI

人工智能，是让机器表现出智能行为的方法与系统总称。

## ML

Machine Learning，机器学习，让机器从数据中学习规律。

## Deep Learning

深度学习，以多层神经网络为核心的方法。

## LLM

Large Language Model，大语言模型，擅长理解和生成自然语言。

## Token

模型处理文本时使用的基本单位，不完全等于“字”或“词”。

## Prompt

给模型的输入指令与上下文。

## Context Window

模型一次能处理的上下文长度上限。

## Hallucination

幻觉，指模型生成了看似合理但并不真实或不可验证的内容。

## RAG

检索增强生成，先检索外部知识，再结合模型生成答案。

## Agent

围绕目标进行多步执行、并能调用工具的 AI 系统。

## Fine-tuning

在基础模型上继续训练，使其更适合特定任务。
