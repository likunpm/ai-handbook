# 大语言模型 LLM

LLM，Large Language Model，中文通常叫“大语言模型”。

它是一类在超大规模文本数据上训练出来的模型，核心能力是理解和生成自然语言。

## LLM 在做什么

从底层看，LLM 的核心任务可以粗略理解为：

“基于上下文，预测下一个最可能出现的 token。”

但只用这句话理解它会过于狭窄。因为当模型规模、训练数据和训练方法足够强时，这种预测能力会表现为更复杂的能力，例如：

- 问答
- 总结
- 翻译
- 代码生成
- 信息提取
- 文本改写
- 多轮对话

## 为什么 LLM 会带来巨大变化

过去的软件系统往往要求用户适应系统结构化输入。

LLM 让系统开始能直接处理自然语言，因此交互门槛大幅下降。它的价值不只是“聊天”，更是把自然语言变成一种新的软件接口。

## LLM 的典型优势

- 通用性强，一套模型可覆盖多种任务
- 上手快，很多场景通过 Prompt 就能试验
- 对非结构化文本处理能力强
- 可以作为其他系统的推理与交互中枢

## LLM 的典型局限

- 可能出现幻觉，即生成看似合理但不真实的内容
- 对实时信息与私有知识并不天然掌握
- 输出稳定性有限
- 成本、延迟和上下文窗口都有约束

## 它为什么是 AI 产品时代的核心

因为 LLM 让很多原本需要单独建模的任务，被统一到“语言理解与生成”这个入口下。

这意味着很多产品形态会发生变化：

- 搜索会变成问答与研究助手
- 表单会变成对话式输入
- 工作流会变成“人 + 模型 + 工具”的协作流程

## 一句话理解

LLM 是能把自然语言当作通用接口来理解、生成和组织任务的大规模模型。
